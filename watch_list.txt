# 자사상품 워치리스트 — 한 줄에 1개씩, 두 가지 방식 혼용 가능
#
# 1) 상품코드(숫자만): 해당 상품이 랭킹에 있으면 매칭
#    예) 1083259765
# 2) 브랜드명(텍스트): 상품의 브랜드 또는 점포명에 포함되면 매칭 (대소문자 무시)
#    ※ 큐텐에 표기되는 이름(영문/가타카나)으로 쓸 것 — 한글 브랜드명은 매칭 안 됨
#    예) innisfree
#    예) イニスフリー
#
# 매칭된 상품은 CSV의 '자사상품' 컬럼에 Y로 표시되고,
# 매칭이 1개라도 있으면 그 시각 랭킹 페이지 스크린샷을 추가 저장합니다.
